# Sunshine Streaming

## Informacion
- Version: 2025.924.154138
- Publisher: LizardByte
- Binario: /usr/bin/sunshine

## Configuracion
Archivos en: ~/.config/sunshine/
- apps.json
- sunshine.conf
- sunshine_state.json

## Autostart
Sunshine arranca automaticamente con XFCE.
Archivo: ~/.config/autostart/sunshine.desktop

## DPMS Deshabilitado
Para evitar que se apague la pantalla:
Archivo: ~/.config/autostart/disable-dpms.desktop

## Acceso
WebUI: https://localhost:47990
Puerto streaming: 47989

## Pairing
1. Abrir Moonlight
2. Anadir PC: 192.168.1.55
3. Introducir PIN
4. Seleccionar Desktop
